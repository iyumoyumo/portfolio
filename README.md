##使用技術
- Laravel
- React
- MySQL

## 機能一覧
- 社員登録
- 社員一覧
- 売上登録
- 売上一覧
- 売上比較(個人別・週・月・年）グラフ表示
- 売上比較（全体・週・月・年）グラフ表示

- iyumoyumo/sales-backend（Laravel）

## 工夫した点
売上管理アプリを作成しました。社員毎に売上を登録、売上を週、月、年で集計して下部でグラフ表示が出来るようにしています。
今回のポートフォリオ制作では、React・Laravel の理解を深めるために AI を積極的に活用し、コード生成や設計の補助として利用しました。
AI の提案をそのまま使うのではなく、自分で読み解き、必要な修正や改善を加えながら完成させています。
ローカルで動く物を作成しGithubにプッシュするところまでは割と早く出来ましたが、公開用のUrlを作成するための準備に時間を割きました。
最終的には、React(Vercel),Laravel+Mysql(Railway)で公開用のUrl作成に成功しました。
## 今後の改善点
タクシー会社で勤務中ですので配車システムまたは工程作成アプリを作成したいと思っています。今回作成した機能は、全体の中の一部として取り込みたいと思っています。
