# 売上管理アプリ (Portfolio)

## 概要
Laravel + React + MySQL で作成した売上管理アプリです。

##使用技術
- Laravel
- React
- MySQL

## 機能一覧
- 売上登録
- 売上一覧
- API 連携
- 売上比較(個人別・週・月・年）グラフ表示
- 売上比較（全体・週・月・年）グラフ表示

## リポジトリ構成
- iyumoyumo/sales-frontend（React）
- iyumoyumo/sales-backend（Laravel）

## ローカルでの起動方法
- npm run dev
- php artisan serve
- xamppでApacheとMysql start
- http://localhost:5173
